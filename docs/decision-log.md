- **검토한 대안**:
  1. 조건부/권장 위치 — (A) 조건부는 100점에 포함, 권장은 별도 보너스 (B) 전부 100점에 통합 → **A 채택**(사용자 선택). 권장은 추가로 "점수화 없이 참고 표시만"으로 확정(보너스 점수 자체를 만들지 않음)
  2. 카테고리 배분 — (A) 항목수 비례(가27·나13·다33·라7·조건부20) (B) 논문 강조점 반영(가18·나12·다35·라20·조건부15) → **B 채택**(사용자 선택, 추천안)

- **최종 배점**: 가18 · 나12 · 다45 · 라10 · 조건부15 = 100 (권장 4개는 배점 없이 "n/4 충족" 배지)
- **검증**: 사용자가 "논문 내 사례에 적용해보고 효과성이 떨어지면 수정할 가능성을 열어두자"고 명시. 표10(10개 기업 × 12개 필수항목, ○/△/✕)을 ○=raw2·△=raw1·✕=raw0로 근사 변환해 조건부 제외 85점 만점으로 시범 계산한 결과:
