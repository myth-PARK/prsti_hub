# PRSTI 채점 방법론 (Phase 0 초안)

**상태**: 초안(draft) — 배점은 아직 미확정. 그 외 핵심 쟁점 4건은 2026-07-20 사용자 확정(`docs/decision-log.md` DEC-002·005·006·007, `_workspace/rubric_gaps.md` 참고): 영역 구조는 논문 원 카테고리 유지, 논문 미출처 3항목은 제외, 조건부 필수항목의 "조건 비해당 명시"는 만점(auto_max) 처리, 근거는 발행회사 자신의 공시만 인정(제3자 분석 불인정).
**작성 기준**: 최아리·고민서·박신화·이규진·현정훈(2026), 「주가수익스와프(PRS)의 공시 투명성 제고 방안」, 한국회계학회지 35(3): 241-280
**원천 데이터**: [.claude/skills/prs-rubric-traceability/references/paper-items.md](../.claude/skills/prs-rubric-traceability/references/paper-items.md)

이 문서는 `rubric-architect` 에이전트의 절차([prs-rubric-traceability](../.claude/skills/prs-rubric-traceability/skill.md))에 따라 논문의 19개 통합 공시 항목(필수 12·조건부필수 3·권장 4)을 0/1/2점 판정기준으로 옮긴 결과물이다. 모든 항목은 `source_table`/`source_item_id`로 논문 표6~8까지 추적 가능하다.

## 척도에 대한 중요한 안내

논문 표10은 실증분석용으로 ○(충분공시=1점)/△(부분공시=0.5점)/✕(미공시=0점)라는 3단계 척도를 이미 사용했다. 아래의 0/1/2점 척도는 **그 척도를 그대로 가져온 것이 아니라 PRSTI가 새로 설계하는 척도**다 — 논문 원 프롬프트(5절)가 요구한 "0점(미공시)/1점(불완전)/2점(구체적·검증가능)" 정의를 각 항목에 적용한 것이며, 두 척도의 대응관계(○→2, △→1, ✕→0)는 완전히 일치하지 않을 수 있다. 항목별 판정기준은 논문 표10의 실제 채점 근거(무엇이 ○/△/✕를 갈랐는가)를 참고했지만, 최종 문구는 PRSTI 고유의 해석이다.

## 가. 기본 거래 정보 (필수항목 1~4)

### 필수-01. 계약규모
- **0점**: 계약 금액이 공시되지 않음
- **1점**: 계약 금액은 명시되나, 부채총계 대비 비율 등 규모를 판단할 비교 기준이 없음
- **2점**: 계약상 약정 금액 + 기초자산 정보 + 부채총계(장부가) 대비 비율(별도·연결)까지 함께 공시됨
- **인정 예시**: 표10에서 10개 기업 전원이 ○(계약규모 100% 공시) — 논문 표2의 "계약규모(원)" 열에 10개 기업 모두 구체적 금액이 확인됨(예: LG화학 2조 원, SK이노베이션 1조5,000억 원)
- **반례**: 해당 없음(전원 공시)

### 필수-02. 계약당사자
- **0점**: 거래상대방·PRS발행회사 관계가 언급되지 않음
- **1점**: 거래상대방 명칭 등 일부만 언급되고 SPC/금융기관 여부나 PRS발행회사-대상회사 관계 설명이 없음
- **2점**: 거래상대방(SPC/금융기관/주관사) 명칭 + PRS발행회사 + PRS발행회사와 대상회사 간 관계가 모두 명시됨
- **인정 예시**: 표10에서 SK이노베이션·효성화학·롯데케미칼·CJ ENM·넷마블·두산로보틱스·금호건설 = ○
- **반례**: 표10에서 한화솔루션·에코프로 = ✕(미공시), LG화학 = △(부분공시)

### 필수-03. 만기일
- **0점**: 만기일 관련 언급이 없음
- **1점**: 계약기간(예: "3년")은 언급되나 구체적 만기 연월일이 없음
- **2점**: 구체적 만기일(연월일)이 명시됨
- **인정 예시**: 표10에서 한화솔루션·SK이노베이션·롯데케미칼·CJ ENM·두산로보틱스·금호건설 = ○
- **반례**: 표10에서 LG화학·에코프로·효성화학·넷마블 = ✕(미공시)

### 필수-04. 기초자산 상세 정보
- **0점**: 기초자산이 무엇인지 확인할 수 없음
- **1점**: 기초자산 종목명만 언급되고 거래 형태(유상증자/주식매수도)나 거래량·발행가액이 불명확함
- **2점**: 거래 형태 + 거래량(금액, 전체 자본 장부가 대비 비율) + 발행가액이 모두 공시됨
- **인정 예시**: 한화솔루션 — "종속기업인 Q Energy Solutions SE 보통주 2,784,293주(22.65%)를 제3자에 매각"(논문 표4, 부록 A1) — 수량·지분율까지 구체적
- **반례**: 표10에서는 전원 ○이나(형식적 기초자산 명시는 있음), 이는 "종목명 확인 가능" 수준이 대부분이라 실제로는 본 항목의 2점 기준(거래량 대비 비율까지)을 만족하는지 원문 재확인이 필요 — **rubric_gaps.md 항목 참고**

## 나. 정산 및 회계처리 (필수항목 5~6)

### 필수-05. 정산 방식
- **0점**: 정산 방식(현금/주식) 언급 없음
- **1점**: 현금 정산이라는 사실만 언급되고 주당 기준가·정산 시점(만기일시/분기별)이 불명확함
- **2점**: 현금/주식 정산 여부 + 주당 기준가 + 정산 시점이 모두 명시됨
- **인정 예시**: 표10에서 10개 기업 전원 ○
- **반례**: 해당 없음(전원 공시) — 다만 "정산 방식이 언급된다"와 "주당 기준가까지 구체적이다"는 다른 수준이므로, 표10의 ○ 판정을 그대로 2점으로 승격하지 말고 원문 재확인 필요(**rubric_gaps.md**)

### 필수-06. 회계처리 근거
- **0점**: 회계처리 방식·근거 언급이 전혀 없음
- **1점**: "파생상품으로 처리"처럼 분류만 언급되고 K-IFRS 조항 인용이나 판단 근거 서술이 없음
- **2점**: 모회사·대상회사의 회계처리 방식과 근거(K-IFRS 조항 인용, 분류 판단 근거)가 함께 명시됨
- **인정 예시**: 논문 표9 모범안(가상 사례) — "그린인베스트먼트(모회사) 측: 파생상품 자산/부채... K-IFRS 제1109호(금융상품) 준용... 판단 근거: 본 PRS 계약은 형식상 유상증자와 파생상품의 결합이나, 모회사의 IRR 보장으로 인해 실질적 차입거래 성격을 가집니다. 따라서 파생상품을 금융부채로 분류하였습니다." — 이 예시가 2점의 기준선
- **반례**: 표10에서 두산로보틱스·금호건설을 제외한 8개 기업 전원 ✕(미공시). 이 항목이 논문의 핵심 문제의식(회계처리 근거 실종)과 직결됨

## 다. 투명성 강화: 부외부채 (필수항목 7~11)

### 필수-07. 부외부채 재분류 위험 명시
- **0점**: 언급 없음
- **1점**: 파생상품 분류라는 사실만 있고 "해석 변경 시 재분류될 수 있다"는 위험 서술이 없음
- **2점**: "회계기준 해석 변경 시 차입성으로 재분류 가능"과 같이 재분류 가능성과 그 조건이 구체적으로 서술됨
- **인정 예시**: 논문에서도 실제 공시 사례를 찾지 못함(표10 전원 ✕) — accepted_example 없음, `not_found`가 정상적 결과
- **반례**: 표10에서 **10개 기업 전원 ✕(미공시)** — 논문이 지적하는 가장 심각한 공백 항목

### 필수-08. 부채 분류시 영향액(정량)
- **0점**: 부채로 분류될 경우의 영향에 대한 정량 정보 없음
- **1점**: 정성적으로만 "부채비율에 영향을 줄 수 있다"고 서술되고 수치가 없음
- **2점**: 분류 시 부채비율 및 변화율(%p)이 장부가·연결 기준으로 정량 제시됨
- **인정 예시**: 논문 부록 표A3(LG화학 사례)이 유사한 형태의 정량 분석("조정 전 부채비율 110.7% → PRS 계약 후 100.2%")을 보여주지만, 이는 **신용평가사의 외부 분석**이지 LG화학 자사 공시가 아니다 — 이 구분을 rubric_gaps.md에 남긴다
- **반례**: 표10에서 10개 기업 전원 ✕(미공시)

### 필수-09. PRS 발행회사 신용등급 및 재무비율
- **0점**: 신용등급·재무비율 언급 없음
- **1점**: 신용등급 또는 재무비율 중 하나만 언급되거나, 발행회사가 아닌 기초자산(자회사)의 정보만 있음
- **2점**: PRS 발행회사(모회사) 기준 신용등급 + 부채비율 + 이자보상배수 등이 함께 명시됨
- **인정 예시**: 두산로보틱스 = ○(표10)
- **반례**: 나머지 8개 기업 = △(부분공시, 대부분 자회사 정보만 확인 가능), 금호건설 = ✕

### 필수-10. 셀다운(유동화 조건)
- **0점**: 셀다운 관련 언급 없음
- **1점**: 셀다운 가능 여부만 언급되고 제약 조건(제한기간·수량 등)이 불명확함
- **2점**: 가능 여부(완전가능/조건부가능/불가능 3단계 구분) + 제약 조건이 구체적으로 명시됨
- **인정 예시**: 한화솔루션 = ○(표10 유일한 공시 사례)
- **반례**: 나머지 9개 기업 전원 ✕ — LG화학은 특히 "PRS 물량의 셀다운을 원치 않는다"는 입장을 증권사에 밝혔음에도 공시에는 반영되지 않음(논문 부록 사례2)

### 필수-11. 콜옵션 조건
- **0점**: 콜옵션(조기 정산) 관련 언급 없음
- **1점**: 조기 정산이 가능하다는 사실만 있고 시점·가격기준·행사가능기간이 불명확함
- **2점**: 조기 정산 유무 + 시점 + 가격 기준 + 행사 가능기간 연장 가능성이 모두 명시됨
- **인정 예시**: 논문에서 실제 기업 공시 사례를 찾지 못함(표10 전원 ✕)
- **반례**: 표10에서 **10개 기업 전원 ✕(미공시)**

## 라. 거래 구조 분류 (필수항목 12)

### 필수-12. 만기 시 기초자산 재매입 조건
- **0점**: 재매입 의무 관련 언급 없음
- **1점**: 재매입 가능성이 언급되나 대상(전부/일부)·가격기준(고정가/시장가/기준가)·시점(만기일/조기상환일) 중 다수가 불명확함
- **2점**: 재매입 의무 유무(부채성/자본성 판정) + 대상 + 재매입 가격 + 시점이 모두 명시됨
- **인정 예시**: 논문에서 실제 기업 공시 사례를 찾지 못함(표10 전원 ✕)
- **반례**: 표10에서 **10개 기업 전원 ✕(미공시)** — 논문이 신설을 제안한 핵심 항목이자 부채성/자본성 분류의 근간

> **항목 7·8·11·12는 10개 기업 전원 미공시.** 판정기준을 작성할 때 이 4개 항목이 "거의 항상 0점"으로 수렴할 것을 감안해야 한다. 향후 실제 채점에서 이 항목들의 구분력(다른 기업 간 차이를 드러내는 능력)이 낮게 나올 수 있음을 Phase 0 검증 단계에서 함께 확인해야 한다.

## 조건부 필수항목 (3개) — N/A 처리가 핵심 쟁점

### 조건부-01. 가격 보전 조건(PRS이자율)
- **발생 조건**: IRR 또는 최저 수익률 보장 조건이 계약에 존재할 때
- **0점**: 조건이 존재함에도 언급이 없음
- **1점**: "수익률을 보장한다"는 사실만 있고 구체적 보장 수익률·스프레드·지급방식이 없음
- **2점**: IRR 보장 여부 + PRS 보장 수익률 + 회사채 신용등급 대비 스프레드(bp) + 지급방식(고정/변동/만기지급)이 모두 명시됨
- **N/A 케이스**: 논문 5.2.1절은 "본 PRS 계약은 가격 보전 조건이 없으며, 순수 기초자산 가격 변동에 따른 차액 정산 구조"라는 명시적 진술도 유효한 공시로 인정한다. **이 경우를 0점(미공시)과 같이 취급하지 않는다** — 조건 자체가 없음을 명시한 경우 만점(auto_max) 처리한다(2026-07-20 사용자 확정, DEC-005).
- **인정 예시**: 논문 3절 실제 사례 — "SK이노베이션(모회사)은 자회사 SK온의 유상증자에 참여한 투자자에게 연 5%의 IRR을 보장하는 계약을 체결"(2026-07-21 수정, DEC-015 — 기존에 적었던 "정기적으로 매년 4% 지급(PRS 이자)과 만기 일시 지급 0.8%로 병합하여 이행"은 논문 5.2.1절이 "예를 들어..."라며 제시한 가상 예시 문구였고 실제 사례가 아니었음을 발견해 교체)

### 조건부-02. 계약체결목적 및 자금사용처
- **발생 조건**: 자금사용처가 명확하고 특정 목적이 존재할 때
- **0점**: 목적·사용처 언급 없음
- **1점**: "재무구조 개선 목적"처럼 목적만 있고 구체적 사용처(투자/채무상환/운영자금)·사용 예정 시기가 없음
- **2점**: 자금 사용처 + 사용 예정 시기가 구체적으로 명시됨
- **인정 예시**: 논문 부록 표A2(SK이노베이션 유상증자 자금 이용 결과) — 채무상환/운영자금 액수까지 구분된 사례

### 조건부-03. 외부평가기관 검토 기록
- **발생 조건**: 독립적 평가 또는 감사가 수행되었을 때
- **0점**: 평가 수행 여부 자체가 불명확함
- **1점**: 평가를 받았다는 사실만 있고 평가기관명·평가등급이 없음
- **2점**: 평가기관명 + 검토 내용 + 평가 등급이 모두 명시됨
- **참고**: 필수-09(발행회사 신용등급)과 혼동하지 않는다 — 09는 "회사 자체" 신용도, 이 항목은 "PRS 거래 구조 자체"에 대한 평가로 서로 다른 대상이다(논문 5.2.1절)
