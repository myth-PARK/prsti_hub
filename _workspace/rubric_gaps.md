## ~~1. 영역(area) 구조를 어떤 것으로 할지~~ → 확정 (DEC-002)

**답변: (A) 논문 원 카테고리 그대로 유지.** 아래는 당시 검토 내용(기록 보존).


지금 `rubric.yaml`은 **논문 원 카테고리를 그대로** 썼습니다(가.기본거래정보 / 나.정산및회계처리 / 다.투명성강화 / 라.거래구조분류 / 조건부 / 권장 — 총 6개 그룹). 이전 대화에서 검토했던 기획초안의 **5영역·100점안**(거래식별성/계약구조/회계처리투명성/위험정보/정량성과비교가능성)은 채택하지 않았습니다 — 그 5영역 중 "PRS 명칭"·"공시 위치"·"진성매각 판단" 3개 하위 항목이 논문 표6~8에 직접 근거가 없었기 때문입니다(3절 참고).

**선택지**
- (A) 논문 원 카테고리 그대로 6개 그룹 유지 — 추적성은 가장 강하지만 "투자자 판단 순서"라는 기획 의도는 약해짐
- (B) 5영역 재구성을 유지하되 3개 미출처 항목만 제거/대체 — 기획 의도는 살리지만 어떤 항목으로 대체할지 추가 결정 필요
- (C) 다른 재구성안

## ~~2. 조건부 필수항목의 "조건 비해당(N/A)" 처리 방식~~ → 확정 (DEC-005)

**답변: (B) 만점 자동 부여(auto_max).** 아래는 당시 검토 내용(기록 보존).

논문은 "가격 보전 조건이 없음"을 명시하는 것도 유효한 공시로 인정합니다(5.2.1절). 세 조건부 항목(조건부-01/02/03) 모두에 해당하는 문제입니다.

**선택지**
- (A) 분모에서 제외(exclude_from_denominator) — 해당 기업의 영역 최대점수 자체가 줄어듦
- (B) 만점 자동 부여(auto_max) — "명시적으로 없다고 밝힌 것"을 최선의 공시로 간주
- (C) 별도 척도(예: N/A를 3번째 결과로 두고 영역점수 계산에서 분리 표시만)

## ~~3. 외부 분석 자료를 "공시 근거"로 인정할지~~ → 확정 (DEC-007)

**답변: 인정하지 않음. 발행회사 자신의 공시만 인정.** 아래는 당시 검토 내용(기록 보존).

필수-08(부채 분류시 영향액)의 후보 예시로 논문 부록 표A3(LG화학)이 있는데, 이건 **한국신용평가의 분석**이지 LG화학 자사 공시가 아닙니다. 논문 표10도 이 항목을 LG화학 포함 전원 ✕(미공시)로 채점했습니다 — 논문 스스로도 외부 분석은 "공시"로 인정하지 않은 것으로 보입니다.

**질문**: PRSTI도 "발행회사 자신의 공시"만 인정하고, 신용평가사·언론 등 제3자 분석은 근거로 채택하지 않는 것으로 확정해도 될까요? (원 프롬프트 6.2절의 "공시 문서명"을 요구하는 취지상 이게 맞다고 판단되지만, 확인 부탁드립니다.)

## ~~4. 논문 미출처 3항목 포함 여부~~ → 확정 (DEC-006)

**답변: 완전히 제외.** 아래는 당시 검토 내용(기록 보존).

- PRS 명칭 (4.2절: 문제로만 지적)
- 공시 위치 (5.2.3절: 지침의 설계 원칙이지 채점 항목 아님)
- 진성매각 판단 (3.2.1절: 필수-06의 하위 판단 요소)

**질문**: 이 3개를 별도 항목으로 신설할지, 아니면 완전히 제외할지, 혹은 다른 형태(예: "공시 위치"는 채점 항목이 아니라 보고서 전체의 가산/감산 요소로 별도 처리)로 반영할지 결정 필요합니다.

## ~~5. 표10 요약과 실제 원문 상세도의 간극~~ → 채택(기본값, 이견 없어 확정)

**답변: 표10은 참고·검증용으로만 쓰고, 실제 채점 시 원문을 직접 재확인.** AI 권고안대로 진행합니다(여러 턴 동안 이견 제기 없었음). 아래는 당시 검토 내용(기록 보존).


필수-01(기초자산 상세정보)·필수-05(정산방식)는 논문 표10에서 10개 기업 전원 ○로 나왔지만, "○"의 실제 정의가 "언급됨"인지 "2점 기준(거래량 대비 비율까지 / 주당 기준가까지)을 만족함"인지가 명확하지 않습니다. `accepted_example`을 표10 하나만 보고 자동으로 2점 처리하면 과대평가할 위험이 있습니다.

**질문**: 실제 채점 단계(evidence-extractor 가동 후)에서 표10 결과를 참고용으로만 쓰고, 최종 점수는 원문을 직접 재확인하도록 못박아도 될까요? (제 판단으로는 이게 맞다고 생각합니다만, 이것도 회계적 판단이 섞일 수 있어 확인 요청합니다.)

## ~~6. 배점(가중치) 확정~~ → 잠정 확정 (DEC-010)

**답변: 가18·나12·다45·라10·조건부15=100, 카테고리 내 균등배분, 권장은 배점 없이 배지 표시.** 표10 기반 시범 계산으로 스프레드(22.9%~45.9%) 확인 완료. 사용자가 "효과성 떨어지면 수정 가능성 열어두자"고 명시했으므로 **최종 확정이 아니라 조정 가능한 기준선**입니다. 자세한 계산과 발견 사항은 `decision-log.md` DEC-010 참고.
